## Usage

1. Read `SKILL.md` for frontend development guidelines
2. Apply when creating/modifying components, routes, or styles in `apps/web/`

Frontend development guidelines for the wingfox monorepo. Covers React 19 + Vite component conventions, TanStack Router file-based routing, Tailwind CSS v4 styling (mobile-first responsive), state management patterns, and Biome code quality rules.

## When to Apply

Reference these guidelines when:
- Creating or modifying React components
- Adding new routes or pages
- Writing styles with Tailwind CSS v4
- Managing state (local, server, or client)
- Ensuring code quality with Biome

## Quick Reference

| Topic | Key Tool |
