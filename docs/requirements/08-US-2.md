# US-2: 選択式クイズ 詳細仕様

## ユーザーストーリー

> ユーザーとして、楽しい選択式の質問に答えることで、自分の好みや性格の傾向を手軽に伝えられる。

---

## 設計思想

選択式クイズは、スピードデーティング（US-3）で使用する **仮想ペルソナ 3 種を生成するための素材データ** を収集することが主目的である。

| ペルソナタイプ | 生成方針 |
| ------------ | -------- |
| タイプ A（共感型 / `virtual_similar`） | 回答プロファイルに最も近い人物像を生成 |
| タイプ B（補完型 / `virtual_complementary`） | 一部の軸で逆方向の人物像を生成 |
| タイプ C（発見型 / `virtual_discovery`） | 大半の軸でユーザーと異なる意外性のある人物像を生成 |

ユーザーへの見せ方は「楽しいマッチングの準備」として設計する。質問文・選択肢はいかなる評価軸も想起させない表現を徹底する。

---

## 質問設計方針

| 設計要件 | 内容 |
| -------- | ---- |
| 問題数 | 全 10 問 |
| 選択肢 | 各問 4 択（単一選択） |
| 想定所要時間 | 2〜3 分 |
| UI | スワイプまたはタップで直感的に回答 |
| カバー領域 | ライフスタイル・価値観・コミュニケーション・行動習慣・恋愛観 |

---

## ペルソナ生成への活用マッピング

> **開発者向け内部ドキュメント** — ユーザーへの表示・開示は禁止。

クイズ回答結果は以下の軸に集約され、US-3 の仮想ペルソナ生成プロンプトに渡される。

| 集約軸 | 参照質問 |
| ------ | -------- |
| 社交性・外向性 | Q1, Q7, Q8 |
| 会話リズム・深度嗜好 | Q2 |
| ユーモアスタイル | Q3 |
| 感情・自己表現スタイル | Q4 |
| 価値観・行動特性 | Q5, Q6 |
| 人間関係・恋愛観 | Q9 |
| 生活行動圏 | Q10 |

### タイプ A（共感型）生成方針

全軸でユーザーの回答に最も近い人物像を生成する。Session 1 のベースライン取得に使用。

### タイプ B（補完型）生成方針

`planning_style`・`introvert_extrovert` など行動軸で逆方向に設定する。`humor_style` は近似値を維持して摩擦を自然に演出する。

### タイプ C（発見型）生成方針

`lifestyle_zone`・`relationship_ideal`・`economic_value` を大きく外し、ユーザーが想定外と感じる背景・趣味を付与する。残りの軸もランダム性を加味して意外性を最大化する。
